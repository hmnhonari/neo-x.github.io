Install Twiki on Ubuntu 12.04

Apr 21, 2013 • Glen B


This is short guide on how to install Twiki ~5.13 on Ubuntu 12.04. This guide has a fix or two that was not present in any other guide I found.

<ol><li>Install all of the relevant dependencies.

<div class="container"><div class="line number1 index0 alt2">sudo apt-get install apache2 libgdal-perl libcgi-session-perl libhtml-tree-perl liberror-perl libfreezethaw-perl libgd2-xpm rcs libapache2-mod-perl2-doc libapache2-mod-perl2 libapache2-mod-php5 latex2html</div></div></li><li> Download Twiki

If installing in the default directory for Apache (/var/www) place the downloaded zip/tar.gz file in /var/www (http://twiki.org/cgi-bin/view/Codev/DownloadTWiki).</li><li>Install Twiki

<div class="line number3 index2 alt2"><div class="line number4 index3 alt1">cp /var/www/twiki/bin/LocalLib.cfg.txt /srv/www/ducklington.org/twiki/bin/LocalLib.cfg </div>chown -R www-data:www-data /var/www/twiki</div><div class="line number4 index3 alt1"></div></li><li>Configure Twiki
Go to this link and have the page generate the configuration needed for your apache and perl configuration.
http://twiki.org/cgi-bin/view/TWiki/ApacheConfigGenerator

save the generated configuration to /etc/apache2/conf.d/twiki.conf</li><li>Restart Apache2<div class="container"><div class="line number1 index0 alt2">sudo /etc/init.d/apache2 reload</div></div></li><li>Allow configuration file to know where the path to the configuration should be put.

Edit /var/www/twiki/bin/LocalLib.cfg:<div class="line number1 index0 alt2">$twikiLibPath = "/var/www/twiki/lib";</div></li><li>Run Twiki config.
Configure your twiki: go to http://localhost/twiki/bin/configure,  setup admin password and paths.</li><li>
